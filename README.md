# foodguard-ai
AI-powered food adulteration risk analyzer designed for lab professionals, enabling detection and severity assessment based on input parameters, with real-time analytics through an interactive dashboard.

Overview

FoodGuard AI is a machine learning-based system built to assist food safety analysts and laboratory professionals in identifying adulteration in food samples and evaluating its severity. The system transforms manual interpretation of lab parameters into a structured, data-driven decision process.

Key Features
Adulteration detection using a Random Forest machine learning model
Severity classification into Low, Medium, and High categories
Interactive real-time analytics dashboard built with Streamlit
Input-driven system tailored for laboratory test parameters
Fast and user-friendly interface for efficient analysis

Tech Stack
Python
Pandas, NumPy
Scikit-learn (Random Forest)
Streamlit
Matplotlib

Dataset
Source: Kaggle
Size: 1000+ samples
Type: Structured dataset containing food quality and adulteration indicators

System Workflow
User inputs laboratory test parameters
Input data is preprocessed and fed into the trained model
The model predicts:
Presence of adulteration
Severity level (Low / Medium / High)
Results are presented through an interactive analytics dashboard

Use Case

This system is designed as a decision-support tool for:

Food testing laboratories
Quality control analysts
Food safety inspectors

It enables faster and more consistent evaluation of food safety by combining machine learning with structured analysis.

Future Scope
Integration with larger and more diverse datasets
Incorporation of explainable AI for model transparency
Extension to image-based adulteration detection
Cloud deployment for scalable access
